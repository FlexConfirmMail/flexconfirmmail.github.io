FlexConfirmMailプロジェクト#

FlexConfirmMailは、フリーソフトウェアの誤送信対策ソリューションです。 企業のメール運用を主なターゲットとして、 宛先を分かりやすくチェックする仕組みを提供することで、 送信ミスを防止することを目指しています。
