FlexConfirmMailプロジェクト#

FlexConfirmMailは、フリーソフトウェアの誤送信対策ソリューションです。 企業のメール運用において大きな課題となる送信ミスの問題を解決します。

Microsoft Outlookのアドインとして導入することで、 今日から誤送信対策を始めることができます。

このページは、Outlook向けの情報です。 Thunderbird向けの情報は FlexConfirmMail :: Thunderbird向けアドオン を参照してください。
