================
クイックスタート
================

システム要件
============

Microsoft Windows
-----------------

- Windows 7
- Windows 8/8.1
- Windows 10
- Windows 11

Outlook
-------

- Outlook 2013
- Outlook 2016
- Outlook 2019
- Microsoft 365 （デスクトップアプリ）

セットアップ
============

- `FlexConfirmMailの最新版インストーラ`_ を入手します。
- インストーラを実行し、ウィザードを完遂させます。
- 最後の画面が表示されたら「OK」をクリックします。

.. _FlexConfirmMailの最新版インストーラ: https://github.com/FlexConfirmMail/Outlook

動作を確認する
==============

- Outlookを起動し、メニューバーの「ファイル」からオプションを選択します。
- 有効なアドオンの一覧にFlexConfirmMailが存在していれば成功です。
