リリーススケジュール#

  • FlexConfirmMailは年に1回、新しいメジャーバージョンを公開します。

  • 次のメジャーバージョンまでは、四半期に一回のペースでマイナーバージョンを公開します。

サポートについて#

フリーソフトウェアプロジェクト#

  • FlexConfirmMailは ISCライセンス で配布しているフリーソフトウェアです。

  • 公開プロジェクトとしてのバグ報告は GitHub Issues で受け付けています。

法人向けサポート#

  • 社内展開時などの法人向けサポートが必要な場合は、開発元のクリアコードが承ります。

  • 契約の費用と特典については 法人のお客様へ を参照ください。

  • 法人向けサポートに関する問い合わせは こちらの問い合わせフォーム からお願いします。
